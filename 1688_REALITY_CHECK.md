## 1688手动搜索指南

### 步骤1: 获取产品图片

从Keepa CSV导出文件复制`Image`列的第一个URL：
```
https://m.media-amazon.com/images/I/71h2vMaS4bL.jpg
```

### 步骤2: 1688以图搜货

1. 打开 https://www.1688.com
2. 点击搜索框右侧的"相机"图标
3. 粘贴Amazon图片URL或上传图片
4. 查看搜索结果

### 步骤3: 选择供应商

建议关注以下指标：
- **价格**: 对比3-5家供应商
- **MOQ**: 是否符合采购计划
- **诚信通年限**: 3年以上更可靠
- **回头率**: >30%说明质量好
- **标签**: "源头工厂"、"深度验厂"

### 步骤4: 填入报告

打开生成的HTML报告，在"采购成本"输入框填入：
```
采购成本: ¥35.50
```

系统自动计算：
```
头程运费: ¥5.40 (450g × 12RMB/kg)
关税: ¥6.14 (15%)
COGS: $6.45 USD
```

## 结论

- 网络/SSL问题

3. 自动完成后续分析

这样既保证了数据准确性，又控制了实施复杂度。
